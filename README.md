Spam SMS Detection using Machine Learning

Project Overview

This project is developed as part of the **CodSoft Machine Learning Internship (Task 4)**.

The objective of this project is to classify SMS messages as **Spam** or **Ham (Not Spam)** using Machine Learning and Natural Language Processing (NLP).



Dataset

- Dataset: Spam SMS Collection Dataset
- Total Messages: 5572
- Classes:
  - Ham (Legitimate)
  - Spam



Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- NLTK
- Joblib



Project Workflow

1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Text Preprocessing
5. TF-IDF Vectorization
6. Model Training
7. Model Evaluation
8. Model Comparison
9. Save Best Model
10. Predict New SMS Messages



Machine Learning Models

- Multinomial Naive Bayes
- Logistic Regression
- Linear Support Vector Machine (SVM)



Model Evaluation

The models were evaluated using:

- Accuracy
- Confusion Matrix
- Classification Report
- Precision
- Recall
- F1-Score
